# 🧘 Gita Detox
A GenAI-powered mental wellness web application inspired by the Bhagavad Gita.

## 🚀 Features
- Provides calming responses based on Gita philosophy
- Simple and clean user interface
- Deployed and accessible online

## 🛠 Tech Stack
- Python
- Streamlit
- Groq API (LLaMA model)

## 🌐 Live App
Access link- https://gita-detox-rm9s3saykhjpwwdb7algcn.streamlit.app/

## 📌 Description
The aim of this project is to help users reduce stress by offering short, meaningful guidance inspired by the Bhagavad Gita.
